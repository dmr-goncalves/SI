Training

Learning in neural networks is mediated by changes in synapses or weights which change how the network processes information. In Simbrain, learning can either happen locally in synapses, in some types of neuron groups, in subnetworks, and in scripts. Learning algorithms for specific synapse and subnetwork types are described in those pages.

This page points to a few training-related components and algorithms that appear in multiple places in Simbrain
